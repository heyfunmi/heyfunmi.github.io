Exploratory Data Analysis With Excel: MonkeyPox Disease.

In this project, I carried out exploratory data analysis on the monkey pox disease. In May this year, several cases of the disease was recorded in various countries around the world, and the disease has continue to garner more attention. So I got this dataset from WHO that contains records of the disease from January 2022 to August 2022, and decided to carry out some analysis on the dataset. Data cleaning, analysis and visualization was done using Excel. Insights like total number of confirmed cases, countries with most cases, and so many more can be seen on the dashboard created.

SALES REPORT ANALYSIS USING POWER BI

Performed exploratory analysis and visualization on a superstore sales records from 2012 to 2017, using Power BI. The Power BI power query editor was used for data cleaning, and visualization was done to show important KPIs, and to get insights which could augment the super store sales record. Analysis of this dataset also provides answers to questions such as the most sold goods by the superstore, most profitable sales channel, least profitable goods and so many more. I also did profer strategic recommendations that could boost the growth of the superstore.

DATA EXPLORATION AND ANALYSIS USING PYTHON

Performed exploratory analysis on gun violence incidents across the fifty states in the USA. I totally enjoyed working on this project as it was a jaw-dropper to me. The dataset contains records of gun violence incidents in the USA from 2018 to 2022. Analysis was done using Python libraries such as Numpy, Pandas, Matplotlib and Seaborn. The analysis of this dataset gives answers to some important questions such as; states with the most gun violence incidents, cities with the most gun violence incidents, total number of deaths caused by gun violence incidents within the period of this record, and answers to a plethora of intriguing questions .
